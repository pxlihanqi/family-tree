# Supabase 照片存储配置指南

为了支持相册功能，需要在 Supabase 控制台中配置照片存储桶。请按照以下步骤操作：

## 步骤 1：登录 Supabase 控制台

1. 打开 [Supabase 控制台](https://app.supabase.com/)
2. 选择您的项目

## 步骤 2：创建照片存储桶

1. 在左侧导航栏中，点击 "Storage"（存储）
2. 点击 "New Bucket"（新建存储桶）按钮
3. 在弹出的对话框中：
   - **Name**（名称）：输入 `photos`
   - **Public**（公开）：勾选此选项，使存储的照片可以公开访问
   - **File Size Limit**（文件大小限制）：建议设置为 10MB，足够存储高清照片
   - **Allowed MIME Types**（允许的 MIME 类型）：输入 `image/*`，只允许上传图片文件
4. 点击 "Create Bucket"（创建存储桶）按钮

## 步骤 3：配置存储桶权限

1. 存储桶创建成功后，点击存储桶名称进入详细设置
2. 确保 "Public" 选项已启用
3. 点击 "Policies"（策略）标签页
4. 点击 "New Policy"（新建策略）按钮
5. 选择 "Allow public access"（允许公开访问）模板
6. 点击 "Use this template"（使用此模板）按钮
7. 点击 "Review"（审核）按钮
8. 点击 "Save Policy"（保存策略）按钮

## 步骤 4：测试存储功能

1. 尝试上传一张照片到存储桶
2. 验证照片是否成功上传并可以通过 URL 访问

## 注意事项

- 确保您的 Supabase 项目已正确配置，并且环境变量已设置
- 存储桶名称必须为 `photos`，否则需要修改代码中的 `PHOTOS_BUCKET` 常量
- 对于生产环境，建议设置更严格的存储桶权限和文件验证

## 故障排除

如果遇到文件上传失败的问题，请检查：

1. Supabase 存储桶是否正确创建和配置
2. 环境变量是否正确设置
3. 网络连接是否正常
4. 文件大小是否超过限制
5. 文件类型是否为允许的图片类型
